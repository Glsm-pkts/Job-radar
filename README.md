# Radar App

Bu proje, uçuşların iniş, kalkış ve varış yerlerini detaylı bir şekilde gösteren bir radar uygulamasıdır.

## Kullanılan Teknolojiler

- **@reduxjs/toolkit**
- **axios**
- **bootstrap**
- **leaflet**
- **moment**
- **react**
- **react-dom**
- **react-leaflet**
- **react-paginate**
- **react-redux**
